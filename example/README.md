# Fat Markdown Demo

A Flutter application demonstrating the usage of the `fat_markdown` package.

## Features

- **Markdown Editor**: A rich text editor for writing and editing Markdown content.
- **Live Preview**: Real-time preview of the rendered Markdown.
- **Tree View**: Toggle to view the internal tree structure of the parsed Markdown.
- **Theme Support**: Switch between light and dark modes.
- **Responsive Design**: Adapts to different screen sizes (mobile and desktop).
- **External Links**: Clickable links that open in the browser.
- **Image Viewer**: Tap on images to view them in a full-screen slider.

## Getting Started

### Prerequisites

- Flutter SDK installed and configured.

### Installation

1. Clone the repository (or copy the example code).
2. Open the project in your Flutter IDE.
3. Run `flutter pub get` to fetch dependencies.
4. Run the application using `flutter run`.
